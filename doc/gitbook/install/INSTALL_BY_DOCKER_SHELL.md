# docker命令一键脚本部署

为了简便大家的部署操作，项目默认提供了docker命令的一键部署脚本，macos, ubuntu都可以用

脚本放在 `bin/` 目录中，脚本名称为 `auto-pull-and-start-docker.sh`

### 进入脚本目录

```
cd bin/
```

### 给脚本执行权限

```
chmod +x ./auto-pull-and-start-docker.sh
```

### 执行脚本

```
./auto-pull-and-start-docker.sh
```
